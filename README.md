# Construct Light

PWA для учёта бюджета строительства — **светлая тема + изумрудно-зелёный акцент** (`#18A558`).
Светлый редизайн форка [ConCalc](https://github.com/demohi6/ConCalc) (тёмная+золото).

🔗 https://demohi6.github.io/ConCalc-light/

Палитра, типографика и спеки компонентов — в [`DESIGN-LIGHT.md`](DESIGN-LIGHT.md).
Эталон-мокап дашборда — [`mockup-dash.html`](mockup-dash.html).

Локальный запуск: `./serve.ps1` → http://localhost:3000
