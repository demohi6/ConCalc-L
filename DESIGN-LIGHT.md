# 🎨 Светлый редизайн Tikinti — бриф для сборки

> Активная инициатива (июнь 2026). Перевод UI с **тёмной+золото** на **светлую тему + изумрудно-зелёный акцент**.
> Этот файл — точка входа для нового чата. Живой мокап: открыть `http://localhost:3000/mockup-dash.html` (через `serve.ps1`).

## Откуда идея
Дмитрий принёс рекламный баннер Orient Academy «Android Kotlin kursuna yazıl». Оттуда взята визуальная ДНК:
- светлый фон, **жирные чёрные заголовки**, одно слово — зелёным («Добрый день, **Дмитрий**» = их «Bir ideyanı **mobil**»);
- изумрудно-зелёный акцент вместо золота;
- мягкие белые карточки с лёгкими тенями, крупные скругления;
- **кольцо прогресса** (их «75% Today's Plan») и **чек-лист** событий с цветными иконками-плитками;
- фото-карточка во всю ширину с именем поверх и бейджем-метрикой в углу (их «Switzerland / 4.8★»).

Светлая тема заодно закрывает роадмап-пункт «Светлая тема — на солнце тёмная плохо видна».

## Типографика
Оставляем нынешние шрифты: **Unbounded** (заголовки/числа-герои), **Mulish** (текст), **DM Mono** (денежные суммы). Везде **sentence case**, без КАПСА (РАСХОДЫ → Расходы).
Открытый вопрос: шрифт баннера плотнее/уже (типа Sora / Plus Jakarta). Можно показать вариант с заменой Unbounded — решали оставить Unbounded как идентичность приложения.

## Компоненты (готовы в мокапе `mockup-dash.html`)
Дашборд = телефон-фрейм → статус-бар → приветствие → **карточка проекта** → точки → **«Сегодня»** → **«Этапы»** → таб-бар.

1. **Карточка проекта — фото-герой** (главный экран). Структура:
   - hero `<img>` высотой ~202px, `object-fit:cover; object-position:center 40%`, скруглён только сверху;
   - скрим: `linear-gradient(to top, rgba(8,20,14,.82), … 0)` снизу — под белый текст;
   - верх-лево: красная плашка «N просрочено»; верх-право: белая пилюля «33% освоено» (это их «4.8★»);
   - низ-лево: имя (Unbounded 19/700, белым) + «Баку · Стройка» с `ph-map-pin`;
   - белый футер: строка «10 этапов · 3 готово / осталось N», полоса прогресса, 3 метрики **Расходы / Остаток / Долги** (долги — красным).
   - **Обложка = последнее фото объекта из журнала** (мини «фото-прогресс»). Нет фото → иллюстрация (нарисованный дом+кран, см. визуал-мокап) или эмодзи-плашка.
   - Альтернатива (первый визуал-мокап): компактная карточка с **зелёным кольцом 33%** слева + 3 метрики. Решение: фото-герой на дашборде, кольцевую — возможно в таб Финансы.

2. **«Сегодня»** — как их «Today's Plan»: заголовок + дата, две пилюли (Потрачено / Бригада), лента событий (иконка-плитка с цветом по типу: note=серый, payment=зелёный, delivery=синий, problem=красный) + разделитель «Ранее» + «Весь журнал» + поле «Что произошло?..» с камерой и зелёной кнопкой отправки.

3. **«Этапы»** — полоса-путь из 10 сегментов (готов=зелёный, просрочка=красный, активный=светло-зелёный `#8FD3AD`, idle=`#E3E8E5`) + герой активного этапа (плитка-иконка, имя, пилюля статуса, бар, факт/смета, дедлайн или «просрочен на N дн»).

4. **Таб-бар** — белый, активный таб зелёный, центральная `+` = зелёный скруглённый квадрат с тенью; снизу home-indicator.

## ⚠️ Реалии внедрения (важно для нового чата)
- **Мокап рисован на Tabler-иконках** (`ti-*`), потому что их требует инструмент visualize. **В реальном приложении — Phosphor (`ph-*`)**. Маппинг: house→ph-house, users→ph-users, wallet→ph-wallet, dots→ph-dots-three, calendar-event→ph-calendar-blank, coins→ph-coins, package→ph-package, note→ph-note-pencil, circle-check→ph-check-circle, camera→ph-camera, send→ph-paper-plane-right, stack-2→ph-stack, alert-triangle→ph-warning, map-pin→ph-map-pin, cloud-rain→(нет в субсете → ph-warning). Локальный шрифт Phosphor — **подмножество**, каждую иконку проверять `getComputedStyle(el,'::before').content`.
- Цвета в приложении не только в CSS-переменных (`style.css`), но и **жёстко зашиты инлайном** в render-функциях `js/*.js` (напр. `rgba(255,255,255,.06)`, золотые хексы, `var(--bg2)`). Реальный перевод = (1) переопределить переменные в `style.css`, (2) пройти инлайн-литералы в js, (3) проверить контраст (белый текст → тёмный), (4) иконки/эмодзи на светлом. Объёмно, но механически. Делать **экран за экраном**, сверяясь с мокапом.
- Стадии-иконки этапов сейчас эмодзи (🏗🧱🪵…) — на светлом читаются, можно оставить.
